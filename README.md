# Нативный виджет лимитов Google AI Pro для macOS (Версия 2.0)

Это самостоятельное нативное приложение для macOS, написанное на Swift и Python. Оно позволяет отслеживать текущие лимиты использования моделей (Gemini и Claude/GPT) для нескольких аккаунтов Google прямо в статус-баре (Menu Bar) операционной системы.

Виджет разработан в премиальном стиле Apple, работает полностью автономно (больше **не требует SwiftBar**) и обладает интеллектуальной системой фонового обновления данных.

---

## 🌟 Ключевые особенности версии 2.0

* **Полная нативность:** Самостоятельное приложение macOS (`GeminiLimits.app`), написанное на Swift (Cocoa). Никаких сторонних плагинов и утилит.
* **Apple-дизайн и выравнивание:** 
  * Все элементы меню (иконки, названия, шкалы лимитов, проценты и таймеры сброса) идеально выровнены по левому краю по единой вертикальной сетке.
  * Активный аккаунт выделяется фирменным зеленым цветом и иконкой галочки.
  * Поддерживается стандартное нативное выделение синим цветом при наведении курсора на строки аккаунтов с клавиатурной навигацией.
* **Оживающий индикатор статуса (Радар):**
  * В режиме ожидания (`idle`) в статус-баре отображается аккуратная белая "точка в тонком круге" в стиле системных индикаторов Apple.
  * При активности ИИ-агента (когда идет диалог) иконка окрашивается в фирменный зеленый цвет Apple (`#30D158`), а от центральной точки начинают плавно расходиться круговые волны радара.
  * При возникновении необходимости подтверждения команды или доступа иконка меняется на предупреждающий треугольник, пульсирующий оранжевым/желтым цветом.
* **Гибридное интеллектуальное обновление квот:**
  * **Умный фоновый таймер:** Каждые 10 секунд приложение проверяет активность ИИ-агента. Если идет активная работа, квоты обновляются раз в **1 минуту** (чтобы вы видели актуальные расходы). Если ИИ-агент отдыхает, обновление снижается до **5 минут** для сохранения батареи Mac и квот Google API.
  * **Мгновенный опрос при открытии:** При клике на иконку в статус-баре приложение мгновенно запускает асинхронный опрос API в фоновом потоке, не вызывая зависаний меню. Данные обновляются прямо в открытом списке.
* **Смена аккаунтов в 1 клик:** Нажатие на неактивный импортированный аккаунт безопасно закрывает Antigravity IDE, переключает OAuth-сессию в базе данных `state.vscdb` на лету, автоматически перезапускает IDE и обновит лимиты.

## 📂 Структура проекта

* `app/main.swift` — Исходный код нативного Swift-приложения на Cocoa.
* `app/compile.sh` — Скрипт для быстрой компиляции Swift-кода в готовый бандл `GeminiLimits.app`.
* `app/Info.plist` — Файл конфигурации для корректного отображения приложения в статус-баре.
* `auth.py` — Скрипт авторизации для добавления новых аккаунтов Google и генерации бинарных слепков сессии.
* `fetcher.py` — Скрипт-сборщик квот. Запрашивает лимиты у Google API и сохраняет их в `quotas.json`.
* `switch_account.py` — Скрипт для безопасного переключения OAuth-сессий в базе данных IDE на лету.
* `generate_sessions.py` — Утилита для генерации слепков сессий для сохраненных аккаунтов.

## 🚀 Настройка и запуск

### Шаг 1. Авторизация аккаунтов
Для каждого аккаунта Google, лимиты которого вы хотите отслеживать, выполните в терминале:
```bash
python3 auth.py
```
1. Скрипт откроет страницу авторизации в браузере.
2. Войдите в аккаунт и дайте разрешение.
3. Введите алиас для отображения в меню (например: *Рабочий*, *Личный*).
4. Токены и слепок сессии будут сохранены в `accounts.json`.

### Шаг 2. Компиляция приложения
Соберите нативное приложение из исходного Swift-кода:
```bash
./app/compile.sh
```
В корне проекта будет создана папка `build/`, содержащая готовое приложение `GeminiLimits.app`.

### Шаг 3. Запуск виджета
Запустите собранное приложение:
```bash
open build/GeminiLimits.app
```
Виджет сразу же появится в статус-баре macOS, сделает первый опрос квот и перейдет в режим фонового мониторинга.
